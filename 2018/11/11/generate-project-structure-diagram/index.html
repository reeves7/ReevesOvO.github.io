  上面这种树形项目结构图在前端项目介绍中非常常见，之前都以为是通过生成器生成或是手敲的，今天才发现linux下使用tree命令就可以直接生成上面的项目结构图。mark一下。

安装

  这里就介绍mac的安装方法，通过homebrew安装就可以了。

brew install tree
# 安装完成之后查看安装成功与否
tree --version

  命令太多？了解了一下其实常用的只有下面着三个：

-d # 只显示目录
-I # 忽略目录或文件
tree -I "node_modules|*.png"
-f # 显示完整路径和文件

  上面的项目结构图我是用以下命令生成的：

tree -I "node_modules"
